Student App

About This App

The Student App is a simple web application designed to help manage student records, attendance, and test marks efficiently. It is built using modern web technologies like Node.js, Express, MongoDB, and HTML/CSS with Bootstrap.

Main Features:

  • Student Registration and Management
  • Admin Panel for Attendance and Marks Entry
  • Email Notifications for Attendance
  • Downloadable Reports

This project was created as a learning tool to understand full-stack development using modern tools and frameworks.
